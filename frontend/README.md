# AI Medical Assistant Application

A production-ready frontend for an AI Medical Assistance Application built with React, Vite, and Bootstrap 5.

## Features

- **Landing Page**: Clean hero section with clear value proposition
- **Medical Assistant**: Chat interface with voice input and AI avatar
- **Medicine Assistance**: Search and information lookup for medicines
- **Appointment Assistance**: Doctor category selection and appointment booking UI

## Tech Stack

- React 18
- Vite
- Bootstrap 5
- React Router DOM

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Design Philosophy

This application follows healthcare UX standards with:
- Professional, minimal design
- Calm and trustworthy color palette
- Large, readable fonts
- High contrast with soft colors
- Accessibility-first approach
